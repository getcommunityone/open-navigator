# Runtime requirements for the Open Navigator web app on Databricks Apps.
#
# SCOPE: serving only. Installs just what `api.main:app` needs to (a) start up and
# (b) serve requests against Neon prod Postgres + the built static SPA/docs.
#
# It deliberately EXCLUDES the heavy data/ML/scraping stack from the repo-root
# requirements.txt (torch, transformers, sentence-transformers, spaCy, mlflow,
# langchain, splink, playwright, selenium, crawl4ai, pyspark, chromadb, qdrant,
# geopandas, datasets, whisperx, ...). Those are imported LAZILY (inside
# functions) by ingestion / analysis / scraper code paths that the served API
# does not exercise at import time, so they are not needed to start or serve. On
# the default Medium (6 GB / 2 vCPU) compute, installing sentence-transformers
# alone drags in the multi-GB torch wheel and blows the 10-minute startup budget.
#
# The internal `packages/*` libraries are NOT pip-installed; they are made
# importable via PYTHONPATH (set in app.yaml). See README.md.
#
# Pins mirror the repo-root requirements.txt where they overlap.
